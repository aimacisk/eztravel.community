## 3 維度風格學習

### 1. 風格特徵

| 維度 | 觀察結果 |
|------|---------|
| **主色調** | 翠綠 `#11D073`（實測 `rgb(17, 208, 115)`）— 非原規格預期的橙紅，以真實數據為準 |
| **字體系統** | Gudea（Latin）+ Heiti（漢字）+ 微軟正黑體（fallback）；設計系統採 Noto Sans TC（TC 標準）|
| **視覺密度** | 高密度資訊卡片布局，促銷 banner 搶眼，旅遊套餐卡片佔主體空間 |
| **色階深度** | 品牌綠從深 `#0C9251` 到淺 `#E7FAF1` 四個層次，中性灰 9 個 tone |
| **邊框/圓角** | 卡片圓角 8–12px，按鈕圓角 4–8px，細邊框 1px `#E8E8E8` |
| **陰影** | 卡片淺陰影 `box-shadow: 0 2px 8px rgba(0,0,0,0.08)` |

### 2. 動線設計

| 動線節點 | 元素 |
|---------|------|
| **進入首頁** | Hero banner（輪播） → 目的地分類列（按鈕群） |
| **瀏覽旅遊產品** | 旅遊套裝卡片格（4 欄 grid） → 標題 / 價格 / 人數 → CTA 按鈕 |
| **促銷感知** | Accent Orange `#FF8B00` 價格標籤 + Accent Red `#FF7777` 折扣徽章 |
| **下單路徑** | 點卡片 → 詳情頁（tabs：行程/費用/規定） → 日期選擇 → 結帳 |
| **導航** | 頂部 nav：首頁 / 主題旅遊 / 機票 / 旅館 / 自由行 |
| **輔助信任** | 星等（Accent Yellow `#FAB617`）、已訂人數徽章、特色標籤 chips |

### 3. 選用動機

| 動機 | 說明 |
|------|------|
| **品牌對齊** | eztravel.community 是 eztravel 品牌的社群平台延伸，必須沿用品牌綠識別色 |
| **目標用戶一致** | 台灣旅遊用戶已熟悉 eztravel 介面語言，降低認知門檻 |
| **設計成熟度** | 原網站 UX 成熟、視覺系統完整，有充分素材可萃取 token |
| **差異化方向** | Community 版本將更偏向「社群 feed + 旅遊日記」，品牌色保留，密度降低、留白增加 |
